# Research Collector

This directory is reserved for local research collector outputs.

Version 1 keeps network research as an operator-triggered step because sources, scraping rules, and site terms vary. Collected strategy notes should be written into:

`obsidian_vault/BTC/01_Strategy_Research`

Valid strategies are extracted into:

`strategies/yaml`

The rest of the lab can then validate, backtest, score, promote, paper trade, and retire them.
